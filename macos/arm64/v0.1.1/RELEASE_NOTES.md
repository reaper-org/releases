Reaper 0.1.1 — macOS arm64

**Install:** download the **DMG** below. Ignore GitHub's automatic "Source code (zip)" and "Source code (tar.gz)" links — those archives are empty placeholders and are not distributable builds.

Drag Reaper.app to Applications, then launch.

### What's new
- **Compiler settings** — compact table with search filter; configure Python, Java, Rust, Go, and 15+ other language runtimes
- **Data directory** — app data now lives in `~/reaper` (repos, workspaces, settings)
- **UI polish** — menubar dropdown fixes, activity strip styling, commit button icons
- **DMG naming** — `reaper-{version}-macos-{arch}.dmg`

**Tip:** opening the .dmg repeatedly mounts a new Finder volume each time — eject old Reaper drives when done.
